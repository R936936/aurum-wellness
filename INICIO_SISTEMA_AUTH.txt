PRIMERA VISITA:
───────────────

1. Usuario abre: http://localhost:3000
2. Sistema verifica: ¿Está autenticado? → NO
3. Muestra: Pantalla de autenticación
4. Usuario ve: Mensaje de Morpheus 💎
   
   "Hola. Soy Morpheus.
    Te estaba esperando.
    Esta es tu última oportunidad.
    Después, no hay vuelta atrás."

5. Usuario elige: "CREAR CUENTA"
6. Completa: Username, Email (opcional), Password
7. Click en: "TOMAR LA PÍLDORA ROJA" 💊
8. Sistema: 
   • Valida datos
   • Crea cuenta
   • Hace auto-login
   • Cierra pantalla de auth
9. Inicializa app con mensaje personalizado:
   
   "Bienvenido de vuelta, [username].
    Tu transformación continúa. Nivel 1, 0 XP.
    Hoy es un nuevo día. Haz que cuente.
    ¿Qué harás hoy para evolucionar?"

10. Usuario empieza su transformación ✨

PASO A PASO:
────────────

1. ABRIR NAVEGADOR
   → Ve a: http://localhost:3000

2. VER PANTALLA DE AUTH
   → Deberías ver el mensaje de Morpheus
   → Logo AURUM WELLNESS en blanco brillante
   → Tabs: "INICIAR SESIÓN" | "CREAR CUENTA"

3. CREAR UNA CUENTA DE PRUEBA
   → Click en "CREAR CUENTA"
   → Username: test
   → Email: test@test.com (opcional)
   → Password: test123
   → Confirmar Password: test123
   → Click "TOMAR LA PÍLDORA ROJA"

4. VERIFICAR QUE FUNCIONA
   → Deberías ver mensaje de éxito
   → Pantalla de auth se cierra
   → Aparece mensaje personalizado:
     "Bienvenido de vuelta, test.
      Tu transformación continúa. Nivel 1, 0 XP."
   
5. PROBAR FUNCIONALIDADES
   → Escribe en el chat: "hola"
   → Deberías recibir respuesta de Morpheus
   → Tu XP debería subir (verlo en header)
   → Escribe más preguntas para ganar XP

6. RECARGAR PÁGINA
   → Presiona F5 o Cmd+R
   → Deberías seguir logueado
   → Tus datos deberían persistir

7. VERIFICAR EN CONSOLA
   → Presiona F12 (Cmd+Option+I en Mac)
   → En consola, escribe:
     window.AurumAuth.getCurrentUser()
   → Deberías ver tu usuario completo

8. PROBAR LOGOUT
   → En consola: window.AurumAuth.logout()
   → Recarga la página
   → Deberías ver pantalla de auth de nuevo

9. HACER LOGIN
   → Username: test
   → Password: test123
   → Click "ENTRAR A LA MATRIX"
   → Deberías recuperar todos tus datos


DEBUGGING:
──────────

Ver todos los usuarios:
  window.AurumAuth.getUsers()

Ver estadísticas:
  window.AurumAuth.getUserStats('test')

Limpiar todo:
  localStorage.clear()
  location.reload()
